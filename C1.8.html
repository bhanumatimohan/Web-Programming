DNS
Stands for Domain Name System, connects URL with their IP Addresses.
TCP
Stands for Transmission Control Protocol, is a connection Oriented Protocol.
HTTP
Stands for HyperText Transfer Protocol, runs on the top of TCP/IP Protocol for file exchange in World Wide Web.
FTP
Stands for File Transfer Protocol, transfer of files from one host to another.
UDP
Stands for User Datagram Protocol, is a connectionless Protocol.
